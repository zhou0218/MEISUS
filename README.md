## 项目概述

该项目是在一个项目的基础上改的, 项目也是毕设的要求. 

以个人美术作品交易的网站, 包括 用户注册/登录, 购买/发布, 下订单/发货, 以及后台作品违规作品投诉等...

本项目拥有后台管理, 对用户/作品/投诉/购物车/订单../进行管理,  能对发布的作品进行审核, 及时对违规的作品进行截断, 并能接受有问题的作品投诉...
